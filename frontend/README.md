# Prod-Commerce Frontend

React + Vite storefront and admin UI for Prod-Commerce.

## Local setup

```bash
npm install
npm run dev
```

By default the app talks to the API gateway at `http://localhost:8080`.
Override it with:

```bash
VITE_API_BASE_URL=http://localhost:8080 npm run dev
```

## Build

```bash
npm run build
```
